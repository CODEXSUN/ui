# CODEXSUN UI

Shared presentation primitives for CODEXSUN Platform and installable applications.

`@codexsun/ui` contains design tokens, generic controls, layouts, and interaction primitives only. Business fields, workflows, routes, and application-specific menus remain in their owning application repositories.

Read `assist/AGENT-GUIDE.md` before changing UI. The local Assist pack defines the
presentation-only boundary and repository release workflow.
